# Ukázková Czechitas Markdown prezentace # Struktura * Každý `code` blok je slide * Code blok zabraňuje auto-formátování které rozbíjí markdown * Podporujeme typy slidů * Pro příklad koukněte do zdroje tohoto slidu # Inverted slide * `class="inverted"` * Má tmavé pozadí a bílý text Inverted standout (`class="inverted standout"`) pro důležité věty # Compact * Kompaktní * `class="compact"` * Když * Více * Řádek * Na * Jedné * Obrazovce # Centered Vycentrovaný obsah odstavců * `class="centered"` # Chování * Změna slidu na šipky, mezerník, backspace * Na klik levým i pravým tlačítkem * Na scroll myší * Když je třeba něco vybrat nebo na něco kliknout (nebo scroll příliš dlouhého sliku), je třeba držet Ctrl nebo Alt * Umíme tisk # Detaily * [Odkaz](https://czechitas.cz) a *em* mění barvu # Obrázky * Měl by se vejít pod nadpis * Když je `class="fullscreen"`, měl by se vejít do celé stránky # Konec * A to je zatím vše co jsem potřeboval pro rychlý vývoj a verzování prezentací
